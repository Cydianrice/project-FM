# project-FM（音乐电台）
## 功能：
1. 已实现：播放/暂停/切换音乐/调节音量/快进/快退/电台分类/滚动歌词
2. 未实现：评论/喜欢
3.未实现功能思路：
* 存储用户信息
* 当用户点击喜欢时把音乐信息存到“我喜欢的”电台中，通过点击“我喜欢的”这个电台节目，用户可以获取自己点过喜欢的音乐）
* 权限设置：评论读的权限public，写的权限只属于创建它的用户
